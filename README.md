# Academic Planner

A comprehensive application for managing and organizing academic coursework, schedules, and planning.

## Features

- Course management and tracking
- Schedule planning and organization
- Academic goal setting and progress monitoring
- Calendar integration
- Assignment and deadline tracking

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/dwilber09-afk/academic-planner.git
cd academic-planner
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```
